# やさしい解説: エッジ検出 (Python 版)

## エッジってなに？

画像のなかで **色や明るさがガクッと変わる場所** のこと。要するに **物の輪郭** や **境界線** です。

身近な例: 黒板に白いチョークで書いた線 ←→ 黒い背景。この境目が「エッジ」。

## どうやって見つけるの？

コンピュータは **となりのマス目との差** をひたすら計算します。

- 左右のマスとぐっと違う → **縦の線** が走っている
- 上下のマスとぐっと違う → **横の線** が走っている

これを画像中のすべてのマスでやって、「変化が大きい場所」だけ白く塗ると輪郭が浮かび上がります。

## 代表的な方法

- **Sobel**: 縦・横方向の変化をそれぞれ計算する。シンプル。
- **Scharr**: Sobel の改良版。より精度が高い。
- **Laplacian**: 「変化の変化」を見る。細かい線にも反応するがノイズに弱い。
- **Canny**: いちばんよく使われる。ぼかし→勾配→細線化→ヒステリシス閾値の順で高品質なエッジを出す。

## Python での注意点: 型とオーバーフロー

Sobel の結果は **負の値を含む** ことがあります。そのまま `uint8` に格納すると負の値が折り返してしまいます。

```python
# 悪い例 (オーバーフローする)
Gx = cv2.Sobel(gray, cv2.CV_8U, 1, 0)   # CV_8U はダメ

# 良い例 (float64 で保持してから変換)
Gx = cv2.Sobel(gray, cv2.CV_64F, 1, 0)
Gx_abs = cv2.convertScaleAbs(Gx)        # |Gx| を uint8 に変換
```

`cv2.magnitude` で X/Y 成分から大きさを計算する場合も `float64` で扱います。

```python
mag = cv2.magnitude(Gx, Gy)                      # float64
mag_u8 = np.clip(mag, 0, 255).astype(np.uint8)   # NumPy でクリップ → 変換
```

## Canny の閾値の決め方

```python
edges = cv2.Canny(blur, threshold1=50, threshold2=150)
```

- `threshold1` (低い方): これより弱い変化は完全に無視
- `threshold2` (高い方): これより強い変化は確実にエッジ
- その間は、**強いエッジとつながっている** ときだけエッジとして採用

ふつうは `threshold2 ≈ threshold1 × 2〜3` くらいを試してみるのが出発点です。

## 一言まとめ

エッジ検出は **「色が急に変わるところ = 物の境目」を見つける処理**。Python では **`CV_64F` で受け取って `convertScaleAbs` で変換する** 流れを体に覚えさせましょう。
