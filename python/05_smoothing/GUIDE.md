# やさしい解説: 平滑化 (ぼかし) フィルタ (Python 版)

## ぼかすってなに？

各マス目の色を、**まわりのマス目の色と混ぜて** やわらかくすることを「ぼかし (平滑化)」といいます。

身近な例: 写真をすごく拡大すると肌のザラつきが目立ちますね。**周りの色とちょっとずつ平均** をとると、ザラつきが消えてなめらかに見えます。

## ぼかし方は何種類もある

| 名前 | イメージ |
|---|---|
| **ボックス (平均) フィルタ** | まわり 9×9 マスのふつうの平均をとる。シンプル |
| **ガウシアン** | 自分に近いマスほど重く、遠いマスほど軽く混ぜる。**自然な仕上がり** |
| **メディアン** | まわりの値の **中央値** をとる。**ゴマ塩ノイズに超強い** |
| **バイラテラル** | 「色が近いマス」だけと混ぜる。**輪郭を残してぼかす** |

## Python での書き方

```python
# ガウシアンブラー
gauss = cv2.GaussianBlur(src, (9, 9), sigmaX=0)

# メディアンフィルタ (カーネルサイズは奇数の整数のみ)
median = cv2.medianBlur(src, 9)

# バイラテラル (d: 近傍の直径, sigmaColor: 色の重み, sigmaSpace: 距離の重み)
bilateral = cv2.bilateralFilter(src, d=9, sigmaColor=75, sigmaSpace=75)
```

### カーネルサイズは「タプル」で渡す

`GaussianBlur` には `(9, 9)` のように **タプル** で渡します。うっかり `9` と書くと TypeError になるので注意。

## NumPy でノイズを作って比較する

フィルタの効果を確かめるには、NumPy でノイズを加えた画像を自分で作るのが便利です。

```python
import numpy as np
rng = np.random.default_rng(0)
noise = rng.integers(0, 50, src.shape, dtype=np.uint8)
noisy = cv2.add(src, noise)   # saturate（飽和）演算。255 を超えても 255 に丸める
```

`+` 演算子ではなく `cv2.add` を使う理由: NumPy の `+` は **オーバーフローで折り返す** (255+1 = 0) が、`cv2.add` は **255 で飽和** します。画像処理では飽和演算の方が自然です。

## カスタムカーネル

```python
kernel = np.array([[ 0, -1,  0],
                   [-1,  5, -1],
                   [ 0, -1,  0]], dtype=np.float32)
sharpened = cv2.filter2D(src, -1, kernel)
```

`filter2D` の第 2 引数 `-1` は「入力と同じ深度で出力する」という意味です。

## 一言まとめ

ぼかしは **「まわりと混ぜて細かい違いを目立たなくする」** 処理。Python では NumPy でノイズ画像を自作して比較するとフィルタの違いが直感的にわかります。
