# やさしい解説: エッジ検出

## エッジ ってなに？

画像のなかで **色や明るさがガクッと変わる場所** のこと。要するに **物の輪郭** や **境界線** です。

身近な例: 黒板に白いチョークで書いた線 ←→ 黒い背景。この境目が「エッジ」。

## どうやって見つけるの?

コンピュータは **となりのマス目との差** をひたすら計算します。

- 左右のマスとぐっと違う → **縦の線** が走っている
- 上下のマスとぐっと違う → **横の線** が走っている

これを画像中のすべてのマスでやって、「変化が大きい場所」だけ白く塗ると、輪郭が浮かび上がります。

## 代表的な方法

- **Sobel (ソーベル)**: 縦方向と横方向の変化をそれぞれ計算する。シンプル。
- **Laplacian (ラプラシアン)**: 「変化の変化」を見る。細かい線にも反応するけどノイズに弱い。
- **Canny (キャニー)**: いちばんよく使われる方法。次のような **段取り** を踏んで、きれいな線だけ残します。
  1. まずぼかしてノイズを消す
  2. Sobel と同じく変化を計算
  3. 線の上で **いちばん強い場所だけ残す** (細める)
  4. **強い線・弱い線の 2 段階の閾値** で「これは本当に線か?」を判断

## ちょっとしたコツ

- 入力は **グレースケール** にしてから処理するのが基本 (色は不要)
- ノイズが多い画像は、**先にガウシアンでぼかす** ときれいなエッジが取れる
- Canny の「強い閾値」「弱い閾値」は、画像によって調整が必要 (たとえば 80 と 160 など)

## 一言まとめ

エッジ検出は **「色が急に変わるところ = 物の境目」を見つける処理**。Canny を使えば実用的な線がサクッと取れます。
